# Simple-video-editor

A tiny, no-backend **browser-based video editor** built as a single HTML file.  

## How to use

1. Open the page
2. Drop a video file (or click to browse)
3. **Crop**
   - Drag inside the crop area to move it
   - Drag the corners to resize
   - Use **Reset Crop** to go back to full frame
4. **Trim**
   - Drag **Start** and **End** handles to set the range
   - Drag the blue marker to scrub through frames
5. Click **Download** to export
